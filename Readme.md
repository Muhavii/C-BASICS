- This calculator is one of the first things I built to test my C# knowledge. Every now and then, I write random C# lines to stay sharp for Unity Engine, which of course uses C#.
- To Run this, the easiest way is to install dotnet on your computer (mine is brew install dotnet)
and run "dotnet run Calc.cs" on your terminal.
